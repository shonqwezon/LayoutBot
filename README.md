LayoutBot — это Telegram-бот, который помогает разметить данные для использования на [хакатоне](https://github.com/valerialevitskaya1204/hackathon_hse25). Бот позволяет команде эффективно и быстро совместно работать над созданием размеченного датасета.

Особенности
      •  Реализован с использованием aiogram — асинхронного фреймворка для работы с Telegram API.
      •  Данные сохраняются в базе данных MongoDB с возможностью динамического обновления и выгрузки.
      •  Поддержка коллективной разметки с отслеживанием вклада каждого пользователя.
      •  Гибкая настройка правил разметки, адаптированных под задачи хакатона.
      •  Удобный экспорт размеченных данных в форматах JSON.

Технологии
  •  Язык: Python
  Библиотеки:
  •  aiogram — асинхронная работа с Telegram API.
  •  pymongo — управление базой данных MongoDB.
